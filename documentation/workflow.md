# AutoLitRev User Workflow

The literature review process in AutoLitRev follows a structured 7-step workflow, optimized with AI assistance at each stage.

## Step 1: Start Your Research (Input)

The user provides their core research question(s). This is the foundation for the entire review process.
- **Actions:** Define Research Question, Publication Year Range, and Key Search Terms.
- **UI:** Focused input interface with clear parameter definitions.

## Step 2: AI-Powered Query Generation

Based on the research questions, AutoLitRev uses GPT-4o to generate optimized search queries.
- **Actions:** Review generated queries, check rationales, and regenerate if necessary.
- **UI:** Structured cards showing each query's title, the search string, and its rationale.

## Step 3: Corpus Selection & Customization

The system suggests relevant academic databases based on the research context.
- **Actions:** Select databases (PubMed, ArXiv, etc.) and use "Suggest Databases (AI)" for optimized selection.
- **UI:** Database list with descriptions and AI-driven recommendation highlight.

## Step 4: Article Search & Retrieval

AutoLitRev executes the generated queries against the selected databases.
- **Actions:** Start search, monitor real-time progress, and view process logs.
- **UI:** Real-time search log, overall progress bar, and discovery metrics (Total Found, Duplicates Removed).

## Step 5: Screening & Selection

Retrieved articles are presented for review and inclusion.
- **Actions:** Screen articles, view metadata, and check AI confidence scores for inclusion.
- **UI:** Professional table layout with screening stats and AI-assisted filtering.

## Step 6: Structured Review Generation

The system assembles the selected literature into a coherent review based on a chosen methodology.
- **Actions:** Select Review Methodology (Systematic, Scoping, Narrative) and trigger AI synthesis.
- **UI:** Methodology selection panel and a large preview workspace for the generated draft.

## Step 7: Review & Finalize (Editor & Export)

The user can refine the generated review and export the final document.
- **Actions:** Edit the review in a focused environment and export in formats like DOCX, PDF, or Markdown.
- **UI:** Clean rich-text editor with word count metrics and comprehensive export options.
